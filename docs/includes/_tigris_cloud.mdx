### Install Tigris CLI

Use the command below to install the CLI on **macOS**

<Tabs>
<TabItem value="macos" label="macOS">

```shell
brew install tigrisdata/tigris/tigris-cli
```

</TabItem>

<TabItem value="linux" label="linux">

```shell
curl -sSL https://tigris.dev/cli-linux | sudo tar -xz -C /usr/local/bin
```

</TabItem>
</Tabs>

Alternative ways of installation can be found [here](@site/docs/sdkstools/cli/installation.mdx).

### Sign up for Tigris Cloud

You will need to have a Tigris Cloud account to complete this quickstart. If
you don't already have an account, you can

<a href={tigrisConfig.signupUrl}>sign up for Tigris Cloud</a>.

### Login and set up the application credentials

Login to your Tigris Cloud account

```shell
tigris login
```

The next step is to generate credentials to be used by the application we
will be building in this quickstart

```shell
tigris create app_key quickstart "quickstart application"
```

Once the command is executed, it would create the application credentials
(clientID and clientSecret) and print it on the screen. Make a note of `id`
and `secret` from the output.

<details>
<summary>Output</summary>
